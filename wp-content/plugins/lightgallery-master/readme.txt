LightGallery Lightbox Plugin for WordPress
====================================

This plugin will implement the lightgallery lightbox into WordPress

Installation Instructions:

1. Activate the Plugin

How to use this Plugin for Images:

1. Create a Gallery using the Media Manager

Custom Styling for Images:

If you wish to customize the styling of the images, copy the custom.css in the css directory, over to your theme to a file named lightgallery_custom.css, the plugin will take your custom css file over the default one.
